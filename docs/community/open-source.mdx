We are firm believers that Open Source is the key to a long-term sustainable and
prosperous technology community. Shuttle builds on Open Source software, and is
itself Open Source. This means that shuttle is free to use and open to
contributions.

### Contribute

Contributions to improve shuttle are welcome. Contribute to shuttle via
[GitHub](https://github.com/shuttle-hq/shuttle).
